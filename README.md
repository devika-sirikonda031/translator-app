# 🌐 Translator App

A full-stack Translator App built using React and Flask.

## 🚀 Features
- Translate text to Telugu, Hindi, English, French
- Voice output using gTTS
- Clean UI using Bootstrap

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Flask (Python)
- API: Google Translator
- Audio: gTTS

## ▶️ Run Locally

### Backend
cd backend
python app.py

### Frontend
cd frontend
npm install
npm start
